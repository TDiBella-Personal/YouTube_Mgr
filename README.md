# Guitar Lesson Finder

A YouTube search utility for finding guitar instructional videos. Search by song, artist, tuning, and guitar type with results ranked by relevance and prioritizing Carl Brown from Guitar365.

## Quick Start

1. Open `index.html` in your browser
2. Paste a [YouTube Data API v3 key](https://console.cloud.google.com/apis/credentials) into the API Key field (saved automatically for next time)
3. Enter a song name, artist, and preferences
4. Hit **Search Lessons**

## Features

- Search by song name, artist, tuning preference, and electric/acoustic guitar type
- Carl Brown / Guitar365 results are prioritized and highlighted
- Results ranked by: creator match, view count, subscriber count
- Two visual themes: **Vivid** (rock & roll) and **Calm** (high contrast)
- API key persists in browser localStorage
- Responsive layout for mobile and desktop
